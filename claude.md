# Bathera Bathware -- Website Development Guide

## Brand Overview

Bathera is a premium bathware and sanitaryware brand. The website should
communicate luxury, modern ceramic craftsmanship, and professional
manufacturing quality.

Design style must feel: - Premium - Bold - Professional - Dark themed -
Ceramic / bathroom focused - Product showcase oriented

------------------------------------------------------------------------

# Brand Colors

Primary Color #1C969C

Secondary Color #05091E

Additional Colors - #FFFFFF (text) - #E5E5E5 (light text) - #0E1325
(dark surface) - #2FCCD2 (accent highlight)

------------------------------------------------------------------------

# Logo

Logo Path: images/bathera-logo.svg

Images Folder: /images/

------------------------------------------------------------------------

# Technology Stack

The project must use:

-   HTML
-   CSS
-   JavaScript
-   Bootstrap 5.3+
-   GSAP + ScrollTrigger
-   Remix Icons

Use CDN where possible.

# Navigation Menu

Left: Logo

Right: - Home - About - Products - Quality - Export - Contact

Requirements: - Sticky navigation - Background changes on scroll -
Mobile hamburger menu

------------------------------------------------------------------------

# Hero Section

Full width premium hero section.

Features: - Large bathroom product image - Dark overlay - Bold
headline - CTA buttons - GSAP animation

Headline: The New Era of Premium Bathware

Subtext: Discover modern ceramic craftsmanship designed for luxury
bathrooms.

Buttons: - Explore Products - Contact Us

------------------------------------------------------------------------

# About Section

Short brand introduction.

Example content:

Bathera is a modern bathware brand focused on creating premium ceramic
sanitary products that combine design, durability, and innovation.

Products include: - Wash Basins - Table Top Basins - Wall Hung Basins -
One Piece Toilets - Two Piece Toilets - Designer Bathroom Solutions

------------------------------------------------------------------------

# Product Categories

Use Bootstrap grid layout.

Each card contains: - Product image - Category title - Hover animation -
Explore button

Hover effects: - Image zoom - Overlay fade

Suggested categories: - Table Top Basins - Wall Hung Basins - One Piece
Toilets - Two Piece Toilets - Wash Basins - Bathroom Accessories

------------------------------------------------------------------------

# Why Choose Bathera

Icon based features using Remix Icons.

Examples: - Premium Ceramic Quality - Modern Bathroom Design - Durable
Materials - Precision Manufacturing - Global Export Standards

------------------------------------------------------------------------

# Product Showcase

Gallery layout using grid or masonry.

Features: - Large product visuals - Hover animations - Smooth
transitions

GSAP Animations: - Fade in - Parallax scrolling - Image reveal

------------------------------------------------------------------------

# Manufacturing & Quality

Highlight production quality.

Bathera products are manufactured with advanced ceramic technology
ensuring long-lasting performance, elegant finish, and superior
durability.

Focus points: - High-quality glaze - Precision molding - Premium
finishing - Quality inspection

------------------------------------------------------------------------

# Export Section

Highlight international presence.

Bathera proudly serves clients across multiple international markets

------------------------------------------------------------------------

# Contact Section

Include:

Fields: - Name - Email - Phone - Message

------------------------------------------------------------------------

# GSAP Animation Guidelines

Use GSAP + ScrollTrigger.

Hero Section - headline fade + slide up - image reveal

Sections - fade up on scroll - stagger animations

Animation Settings: Duration: 0.8 -- 1.2 seconds Ease: power3.out
